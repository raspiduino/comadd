# ComAdd
This script prevent you from distractions
## Installation
```bash
git clone https://github.com/raspiduino/comadd.git
cd comadd
pip install pywin32
comadd.py
```
Or you can goto <a href="https://github.com/raspiduino/comadd/releases">Release</a>
<br>Don't use the extension yet!
